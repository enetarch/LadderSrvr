Method : CreateTable

Syntax | Description | Parameters | Values | Returns | Remarks | Example | See Also |

Syntax

Private CreateTable (szName, szStructure) : Boolean

Description

Adds a table to the Ladder data store (mySQL).

Parameters

[ szName ] - the name table to be created.

[ szStructure ] - the structure of the table to be created.

Returns

[ Boolean] - True or False on the successful creation of the table.

Remarks

the structure of a table should NOT include the create table keyboards needed by mySQL. Instead only the actual structure needed to maintain the objects instance data should be provided.

Additionally, each table created should not include an ID field. Ladder automatically prepends this to your table, and uses it to maintain the relationships between the instance data and Ladder's Link table.
