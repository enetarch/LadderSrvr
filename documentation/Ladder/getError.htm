Method : get Error

Syntax | Description | Parameters | Values | Returns | Remarks | Example | See Also |

Syntax

Public get Error ( ) : Error

Description

returns an error object encapsulating the error that occured.

Parameters

Values

Returns

[ Error ] - the error, if any, that was captured while processing a request through Ladder.

Remarks

In Visual Basic, the Error object was used extensively to do error tracing. It is not implemented in this version of Ladder at this time to improve the speed of the prototype.
