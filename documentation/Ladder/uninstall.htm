Method : Uninstall

Syntax | Description | Parameters | Values | Returns | Remarks | Example | See Also |

Syntax

Public UnInstall () : Boolean

Description

This removes the Classes, Links, Folders, and Items from the Ladder.

Parameters

Values

Returns

[ Boolean ] - whether the removal was successful.

Remarks

When uninstalled this method removes just the Ladder Links table from the mySQL server. To completely destroy Ladder, issue a drop database and create database.
