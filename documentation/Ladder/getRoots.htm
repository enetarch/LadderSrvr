Property : get Roots

Collection | Syntax | Description | Errors | Events | Examples | See Also | Usage Concepts |

Syntax

Public get Roots ( szName, nClass) : Folders

Description

This returns the root folders found in the Ladder tree.

Parameters

[ szName ] - null or the name of the folder sought after.

[ nClass ] - null or the class of the folder sought after.

Returns

[ Folders ] - the set of Root Folders.

Remarks

In most cases, most applications will only have the Root Folder and Root Classes Folder in the Ladder Tree. Additional Roots might be useful to speed searches, or seperating application modules from each other.
