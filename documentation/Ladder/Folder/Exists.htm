Method : Exists

Syntax | Description | Parameters | Values | Returns | Remarks | Example | See Also |

Syntax

Public Exists ( szName, aryClasses, nBaseType ) : Boolean

Description

Determines if the Folder, Data Item, or Reference are found in this folder

Parameters

[ szName ] - is the Name you are searching for.

[ aryClasses ] - The classes being sought.

[ nBaseType ] - the base type - folder, data item, reference - being sought

Values

0 < szClassName Length < 32

Returns

[ Boolean ] - True if it does, False if it does not.

Remarks

Each Folder, Data Item and Reference can be found by matching it's exact name within the container. If more one or more names match, this function true.

getChildren
getSubFolders
getItems
getReferences
