Syntax

Public getReferences ( szName, aryClasses, nBaseType, nOrderBy, bASC ) : References

Description

Searches the current folder for all the Folder Links that consider this folder to be it's parent.

Parameters

[ szName ] - is the Name you are searching for.

[ aryClasses ] - The classes being sought.

[ nBaseType ] - the base type - folder, data item, reference - being sought

[ nOrderBy ] - order the results by the ID, name, class, base type

[ bASC ] - Sort the results Ascending or Descending [True or False]

Values

Returns

[ References ] - a list of all references found in a References container

Remarks

The ID's of the Links that call this folder parent
