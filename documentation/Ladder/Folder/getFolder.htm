Method : getFolder

Syntax | Description | Parameters | Values | Returns | Remarks | Example | See Also

Syntax

Public get Folder ( szName, nClass ) : Folder

Description

Searches the current folder for all the Folder Links that consider this folder to be it's parent.

Parameters

[ szName ] - is the Name you are searching for.

[ nClasses ] - The classes being sought.

Values

Returns

[ Folder ] - If a folder is found, a Folder Instance is returned
