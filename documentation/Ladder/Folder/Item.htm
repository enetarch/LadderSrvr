Method : Item

Syntax | Description | Returns | Remarks | Example | See Also |

Syntax

Public Item ( nPos, aryClasses, nBaseType ) : [ Folder | Item | Reference ]

Description

searches the current folder for all the items matching the parameter criteria.

Parameters

[ nPos ] - is position of the item of interest in the possible list of elements stored in the folder.

[ aryClasses ] - The classes being sought.

[ nBaseType ] - the base type - folder, data item, reference - being sought

Returns

[ Folder | Item | Reference ] - a container for the Ladder Link returned.

Remarks

Since Ladder doesn't know what the object is until it is retrieved, this function will return one of three (3) objects. In other languages like C# or Java, Ladder would return the Properties object.
