Method : Count

Syntax | Description | Returns | Remarks | Example | See Also |

Syntax

Public Count ( szName, aryClasses, nBaseType ) : Long

Description

Count returns the number of Folders, Data Items and References that are found in this Folder object.

Parameters

[ szName ] - the name of the entries being sought.

[ aryClasses ] - the classes being sought

[ nBaseType ] - the base type - folder, data item, reference

Values

Returns

[ Long ] - the number of items and folder in this folder.
