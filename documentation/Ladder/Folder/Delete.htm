Method : Delete

Syntax | Description | Parameters | Values | Returns | Remarks | Example | See Also |

Syntax

Public Delete () : Boolean

Description

This removes the Folder and it's contents from the Ladder.

Parameters

Values

Returns

[ Boolean ] - depending on whether or not the whole process was successful

Remarks

When a folder is deleted, all it's contents are deleted with it. Presently this is not a two step process, but this could be exteneded to mark the links for deletion and then purge them.
