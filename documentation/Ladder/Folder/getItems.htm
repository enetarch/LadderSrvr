Method : getItems

Syntax | Description | Parameters | Values | Returns | Remarks | Example | See Also |

Syntax

Public getItems ( szName, aryClasses, nBaseType, nOrderBy, bASC ) : Items

Description

Searches the current folder for all the items matching the parameter criteria.

Parameters

[ szName ] - is the Name you are searching for.

[ aryClasses ] - The classes being sought.

[ nBaseType ] - the base type - folder, data item, reference - being sought

[ nOrderBy ] - order the results by the ID, name, class, base type

[ bASC ] - Sort the results Ascending or Descending [True or False]

Values

Returns

[ Items ] - a list of all items found in a Folders container

Remarks

Each ID has a matching position with an instance within the container. You can use this ID value to retrieve the specific instance matching your search criteria.
