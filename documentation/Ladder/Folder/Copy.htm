Method : Copy

Syntax | Description | Parameters | Values | Returns | Remarks | Example | See Also

Syntax

Public Copy ( Folder) : Boolean

Description

Copy, copies all the Folder Items, and References to the target folder.

Parameters

[ Folder ] - is the folder to which all these items are to be duplicated to.

Values

nTarget is not NULL

Returns

[ Boolean ] - True if successful, False if not.

Remarks

Unlike duplicate, which copies all the items and their respective data to the new folder, Copy only copies the references to the data. This places two Ladder Links pointing to the same data element.

Note that this function is not yet implemented in this version of Ladder.
