Method : saveData

Syntax | Description | Parameters | Values | Returns | Remarks | Example | See Also |

Syntax

Public saveData () : Boolean

Description

This saves the state of the Item's Data.

Parameters

Values

Returns

[ Boolean ] - depending on whether or not the instance was successful in transferring it's state to the structure.

Remarks

saveData is called by Store, so unless a specific data update is needed, there is no need to call this function directly. In addtion, if the Data Item has not be saved, calling this function directly will fail.
