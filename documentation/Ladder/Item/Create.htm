Method : Create

Syntax | Description | Parameters | Values | Returns | Remarks | Example | See Also |

Syntax

Public Create ( Folder, szName, szDescription, nClass) : Boolean

Private Create2 ( szName, szDescription, nClass, szTableName) : Boolean

Description

Create initalizes a data item object to the instance data passed in through the parameters. The object will not be stored until STORE is called. That is when the ID initialized.

Parameters

[ Folder ] - the parent link, unless this instance will be a root folder.

[ szName ] - The name of the folder to be created.

[ szDescription ] - The description of the folders contents

[ nClass ] - The class of the folder, item or reference.

[ szTableName ] - This SQL table used by this Data Item.

Returns

[ Boolean ] - True or False if the object was initialized peroperly.

Remarks

The Data Item class extends Ladder's Link objects by connecting them to additional tables that contain data specific to a class, like an address label.
