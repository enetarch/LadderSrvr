Description

A Data Item element stores state data for objects. Where Folders are containers, and References are pointers to folders, Data Item elements allow Ladder to store the state data of objects. Data Item elements have two parts, the Element Link data and the objects state data. They are stored seperately in the mySQL server. The purpose of this is to allow Ladder to organize the state data based on a heirarchial relationship and not be confined to one type of state data format.
