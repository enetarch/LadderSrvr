Method : Update_Item

Syntax | Description | Parameters | Values | Returns | Remarks | Example | See Also |

Syntax

Private Update_Item ( thsState, thsData ) : Boolean

Description

Provides Ladder Client (not yet available) with a quick way to update Data Items.

Parameters

[ thsState ] - this is an associative array contain the Link Instance data.

[ thsData ] - this is an associative array contain the object's Instance data.

Values

Returns

[ Boolean ] - If the update process was successful.

Errors

Table Doesn't Exist - the table that the data is stored in, doesn't exist. The installation process may not have succeeded in creating the table.

Invalid Structure - A field within the structure, generated as a reflection of the table, does not contain a field needed by the instance to store it's information.

Data Type MisMatch - a field that is contained within the structure doesn't match the data type that the Instance needs.

Remarks

Update handles the interaction between the instance and the storage medium.
