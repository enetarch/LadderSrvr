Method : Update_Link

Syntax | Description | Parameters | Values | Returns | Remarks | Example | See Also |

Syntax

Private Update_Link ( thsState ) : Boolean

Description

Provides Ladder Client (not yet available) with a quick way to update links.

Parameters

[ thsState ] - this is an associative array contain the Link Instance data.

Values

Returns

[ Boolean ] - If the update process was successful.

Errors

Remarks

Update handles the interaction between the instance and the storage medium. When an instance needs to be saved, the update contacts the stroage device for a copy of the data structure needed for the instance. Update then passes this structure to the Instance. The Instance then fills in the structure and passes it back to Update. Update then passes it back to the storage device to be saved.
