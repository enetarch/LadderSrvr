Method : Install_Classes

Syntax | Description | Parameters | Values | Returns | Remarks | Example | See Also

Syntax

Private Install_Classes() : Boolean

Description

Ladder has five (5) basic classes that are created when Ladder is installed into a data store. They are the Root Folder Class, Root Classes Folder Class, a generic Ladder Class Folder Class, a generic Ladder Folder Class, and a Ladder Class Definition Class.

Parameters

Values

Returns

[ Boolean ] - whether the install of the storage was successful.
