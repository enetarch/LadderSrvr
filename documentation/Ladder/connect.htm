Method : Connect

Collection | Syntax | Description | Parameters | Values | Errors | Events | Examples | See Also | Usage Concepts |

Syntax

Public Connect ( szODBC) : Boolean

Description

Connects Ladder to and ODBC compliant data store, like mySQL.

Parameters

[ szODBC ] - is the Storage Object that Manages the Database.

Values

HOST - the URL or IP address to the mySQL server

UID - the User ID to pass to the mySQL server for authentication

PSW - the Password to pass to the mySQL server for authentication

DBF - the database in the mySQL server to use

Returns

[ Boolean ] - If the Connection was successful
