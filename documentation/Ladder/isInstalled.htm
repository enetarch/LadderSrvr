Syntax | Description | Parameters | Values | Returns | Remarks | Example | See Also |

Syntax

Public isInstalled () : Boolean

Description

Checks the data store (mySQL) to determine if Ladder has been installed into the target database.

Parameters

Values

Returns

[ Boolean ] - True or False depending on whether or not Ladder has been installed.

Remarks

This does not confirm that Ladder has been installed correctly, just that a table exists.
