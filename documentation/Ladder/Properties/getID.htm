Property : ID

Collection | Syntax | Description | Values | Returns | Remarks | Usage Concepts | See Also

Syntax

Public ID : Long
Friend ID = [Value]

Description

A read only value to non-Friend classes, that returns the ID assigned to the instance from the container class.

Values

set ID assigns the instances it's ID.

Returns

get ID returns the ID of the Instance, as assigned by the container class

Remarks

Depending on the implemention of Ladder, this value may be set by the database engine being used or it may be set by the container class. In either case, this value will be an auto incrementing sequential ID from 1 through infinity.

The set ID is exposed only to the Container, only the container can change the ID of the isntance. Should any other class be allowed to change this value, the engine would loose integrity and crash.
