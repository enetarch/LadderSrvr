Property : getLeaf

Syntax | Description | Returns | Remarks | Example | See Also

Syntax

Public getLeaf : Long

Description

The leaf value can point to either a Link Instance in the Ladder Tree or it can point to a row of data in another table. Which one it points it to depends on the BaseType.

Returns

[ Long ] - the ID of a Link Instance or the ID of another table.

Remarks

Get and Set Leaf should only be called by Ladder. If this value is changed by your application, it could corrupt the tree.
