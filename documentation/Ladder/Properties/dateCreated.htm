Property : Date_Created

Syntax | Description | Returns | Remarks | Example | See Also

Syntax

Public date_Created : Date
Public date_Accessed : Date
Public date_Modified : Date

Description

The Date that the Instance was Created, Accessed, and Modified.

Returns

[ Date ] - the date and time of the event.

Remarks

The Date Created is the actual date that the instance was stored in Ladder. The Date Accessed, is the last date that the Instance was retrieved from the system. (This may or may not be implemented). The Date Modified is the date that the system changed the contents of the instance or the link itself. As if the Link were moved.
