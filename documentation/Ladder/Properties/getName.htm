Property : Name

Syntax | Description | Returns | Remarks | Example | See Also

Syntax

Public Name : String
Public Name= [String]

Description

Name identifies the name of the link and data element to the ladder.

Values

a non-Null String. If a null string is passed it, the name is not changed.

Returns

[ String ] - the name of of the link.

Remarks

The names can be duplicated. Just remember that. Even if the class is the same. That is becuase Ladder uses ID's to properly identify each Link. Through the ID's you can retrieve each link, even if it has the same name.
