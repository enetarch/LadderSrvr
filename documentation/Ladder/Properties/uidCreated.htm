Property : UID_Created

Syntax | Description | Returns | Remarks | Example | See Also

Syntax

Public UID_Created : String
Public UID_Accessed : String
Public UID_Modified : String

Description

The User's ID that the Instance was Created, Accessed, and Modified by.

Returns

[ String ] - the User's ID.

Remarks

Created identifies who created the isntance was stored in Ladder. Accessed, identifies who last retrieved the instance from the system. (This may or may not be implemented). Modified, identifies who changed the contents of the instance or the link itself. As if the Link were moved.
