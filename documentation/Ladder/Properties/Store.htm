Method : Store

Syntax | Description | Returns | Remarks | Example | See Also

Syntax

Public Store () : Boolean

Description

This function focuses solely on insuring the Link Instance's state data is transfered to the store, whether that be a SQL database or some other medium.

Returns

[ Boolean ] - The storage sequence was successful or not.

Remarks

In previous versions of Ladder, all SQL command were pushed through a wrapper class that specifically talked to a particular database engine. In this version, Store expects to communicate to a mySQL database engine.
