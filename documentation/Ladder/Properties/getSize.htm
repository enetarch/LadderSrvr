Property : Size

Syntax | Description | Returns | Remarks | Example | See Also

Syntax

Public Size () : Long

Description

Returns the size of the Link and / or the table data element.

Returns

[ Long ] - The size of the link and / or the table data element.

Remarks

Even though the link's size will remain constant, and the table data element at first glance should be the same size all the time, don't forget that with BLOBs that elements can actually grow up to 2 gig in size.
