Property : GUID

Collection | Syntax | Description | Values | Returns | Remarks | Usage Concepts | See Also

Syntax

Public GUID : String
Friend GUID = String

Description

A read only value to non-Friend classes, that returns a GUID assigned to the instance from the container class.

Values

set GUID assigns a unique String value to this Link instance.

Returns

get GUID returns a String assigned by the container class to this Link instance

Remarks

The GUID is not implemented in this version of Ladder. When implemented, it will allow data between trees to be shared without crashing IDs. In addition, Reference objects will associate to the GUID vs the ID. Presently, the ID of a Link instance can change between trees.

The advantage of switching to GUIDs, is that it speeds data synchronization between mobile devices, as well as allows synchronization between multiple servers.

Note: Some elements within Ladder, like Articles, may embedded GUIDs into their data. While Ladder can see and insure that Folder pointed to by Reference Objects are transfered if needed, it cannot see into other class values to determine if GUIDs are being used there, and cannot insure that their associated data will be available when synchronized.
