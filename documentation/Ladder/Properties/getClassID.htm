Property : ClassID

Syntax | Description | Returns | Remarks | Example | See Also

Syntax

Public ClassID : Long

Description

This returned the ID of the class.

Returns

[ Long ] - the ID of the class associated with this Link Instance

Remarks

Every object in Ladder is of an instance of a class, even the base classes of Folder, Item or Reference. Ladder automatically registers all classes as constants in PHP. (Other programming languages may have different conventions). This allows programmers to determine what class an object is and what steps to take.
