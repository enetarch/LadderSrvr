Property : Connect

Syntax | Description | Parameters | Returns | Remarks | Example | See Also

Syntax

Public Connect ( cn ) : Boolean

Description

This connects Properties Class to the database engine.

Parameters

cn - the database connection object storing Ladder's data.

Returns

[ String ] - the IP Address and time of th event.

Remarks

The IP Created is the actual machine that created the isntance and stored it in Ladder. The IP Accessed, is the last machine that the Instance was retrieved from the system by. (This may or may not be implemented). The IP Modified is the IP of the machine that changed the contents of the instance or the link itself. As if the Link were moved.
