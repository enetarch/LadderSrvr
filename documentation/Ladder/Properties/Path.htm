Property : Path

Syntax | Description | Returns | Remarks | Example | See Also

Syntax

Public Path : Array

Description

Returns

[ Array ] - a set of ID's identifying every parent in the tree for this Link Instance.

Remarks

In any directory tree, it is necessary from time to time to know how you got there. In addition, there are times that the code will need to know who the parent folders and their children are.
