Syntax | Description | Parameters | Returns | Remarks | Example | See Also |

Syntax

Public Delete () : Boolean

Parameters

no parameters are taken

Description

This removes the Link Instance from Ladder.

Returns

[ Boolean ] - stating the success or failure of the deletion.

Remarks

Failure can occure if the Link Instance points to an Item that has not been removed.
