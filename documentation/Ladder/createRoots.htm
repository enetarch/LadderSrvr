Syntax | Description | Parameters | Values | Returns | Remarks | Example | See Also |

Syntax

Private Create_Roots ( ) : null

Description

Creates two roots in the Ladder Links table, during the initialization process.

This function will confirm that the folders do not exist first before creating them, as duplicates will cause problems.
