Property : get / set Folder

Syntax | Description | Parameters | Values | Returns | Remarks | Example | See Also

Syntax

Public get Folder : Folder
Public set Folder ( Long )

Description

get and set Folder allow the application to return or redirect the reference object to point to a folder if needed.

Parameters

[ Long ] - the ID of a Folder Instance.

Returns

[ Folder ] - the Folder pointed to by the Reference instance.
