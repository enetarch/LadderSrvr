Method : Create

Syntax | Description | Parameters | Values | Returns | Remarks | Example | See Also |

Syntax

Public Create ( Folder, szName, szDescription, nClass, LinkTo) : Boolean

Description

Create initalizes a reference object to the instance data passed in through the parameters. The object will not be stored until STORE is called. That is when the ID initialized.

Create Folder, Item, and Reference, add a new instances of Folder, Data Item, or Reference to the current folder. These instances will not be saved to Ladder's store until STORE is called.

Parameters

[ Folder ] - the parent link, unless this instance will be a root folder.

[ szName ] - The name of the folder to be created.

[ szDescription ] - The description of the folders contents

[ nClass ] - The class of the folder, item or reference.

[ LinkTo ] - The folder that this link will be associated with.

Returns

[ Boolean ] - success or failure in initializing the instance.

Remarks

References act like pointers to other folders.
