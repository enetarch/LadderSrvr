Method : get Object

Syntax | Description | Parameters | Values | Returns | Remarks | Example | See Also |

Syntax

Public get Object ( nID, thsObject ) : thsObject

Description

This retrieves a specific element from Ladder's Link table, and initalizes your object with it. This allows you to reuse objects, as long as your code know's what the target object is. Ladder will not ask the target object to validate that it is the right object for the data.

Parameters

[ nID ] - the specific ID of the item to be retrieved

[ thsObject ] - the object you want Ladder to initialize with your data.

Values

Returns

[ thsObject ] - the object passed in, is initialized with the data and returned.
